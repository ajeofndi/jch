아이템매니아 × 라살라스

제안 목적

공동 마케팅으로 유저를 유입시키고, 양사 API 연동 기반으로 성과 흐름을 연결하여 양사 매출을 함께 확대하는 구조입니다.

1. 제안 배경 및 목표

양사는 공동 마케팅과 게임사 API 연동 기반 구조를 통해 쌀먹 유저 및 거래 목적 유저를 라살라스 신규서버로 유입시키고, 이를 바탕으로 라살라스 매출과 아이템매니아 거래수수료 매출을 함께 확대하는 것을 목표로 합니다.

2. 기대효과

라살라스 기대효과

  • 신규서버 초기 유저 유입 및 매출 확대
  • 아이템매니아 유튜버·외부광고·CRM 채널 활용 성과 확대
  • 외부 거래 흐름의 관리 가능 영역화
  • 게임 생태계 활성화 및 고가치 유저 만족도 개선
  • 거래 수수료 추가 수익 발생

아이템매니아 기대효과

  • 라살라스 거래량 증가에 따른 거래수수료 매출 확대
  • 쌀먹 유저 및 거래 목적 유저 유입
  • API 기반 회원들의 간편 거래 지원
  • 오입력·미수령 리스크 완화
  • 게임사 연동형 제휴 사례 확보
